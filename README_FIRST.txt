        THAZEMA - READ THIS FIRST
========================================

IMPORTANT: Node.js is NOT installed on your computer!

You need to install Node.js before you can run this application.

QUICK STEPS:
============

1. Install Node.js
   - Go to: https://nodejs.org/
   - Download the LTS version
   - Run the installer
   - Restart your computer

2. Install MongoDB
   - Go to: https://www.mongodb.com/try/download/community
   - Download and install
   - Start the service: net start MongoDB

3. Install Thazema
   - Open Command Prompt in this folder
   - Run: npm install
   - Run: cd client && npm install && cd ..
   - Run: npm run db:setup
   - Run: npm run db:seed
   - Run: npm run dev

4. Open Browser
   - Go to: http://localhost:3000
   - Login with: abebemesfin53@gmail.com / admin123
